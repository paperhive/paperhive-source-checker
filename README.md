Checks if a given URL is valid for [PaperHive](https://paperhive.org) and
parses it.

## Development
* Check linting:
    ```
    npm run lint
    ```
* Run unit tests:
    ```
    npm test
    ```
* Publish on npmjs.com:
    ```
    npm publish
    ```
* Add Git tag:
    ```
    git tag v2.0.1
    git push --tags
    ```
